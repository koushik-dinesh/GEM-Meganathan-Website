# Gem Consultant Meganathan

Premium, concise corporate website for a GeM marketplace consultancy.

## Stack

- React 19 + Vite
- Tailwind CSS
- Framer Motion + Lenis smooth scroll
- shadcn/ui-style components
- Lucide React

## Sections (~3–4 viewport scrolls)

1. **Hero** — Split layout, founder card, official `gem.png` illustration
2. **Services** — 6 service cards
3. **About** — Consultancy intro + 4 feature cards
4. **Contact** — Info cards, CTAs, Google Maps (no form)

## Assets

- Logo: `/public/gcm-logo.png` (from official site)
- Founder photo: `/public/founder.png`
- GeM illustration: `/public/gem.png`

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
